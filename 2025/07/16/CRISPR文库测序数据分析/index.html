CRISPR文库测序数据分析

Posted on 2025-07-16 Edited on 2025-08-20

分析CRISPR文库的病毒感染效率

制作参考序列文件

通常测序比对时，参考基因组是鼠/人等生物的基因组。
sgRNA测序只需要检测是否转入对应的20 bp sgRNA序列即可，因此参考基因组为设计好的sgRNA序列。
含有sgRNA序列的 excel 示例

参考基因组的文件是FASTA格式，第一行是‘>’加‘染色体名’，后一行是序列。需要手动制作该文件。

比对软件还需要参考文件的索引，制作索引。

分割sgRNA

PCR之后的sgRNA文库，长度为466bp 只需关注20bp的sgRNA部分，位于前150bp内
检查双端测序的结果，只需使用一端测序的文件，另一端测序的结果不包含sgRNA部分。
使用cutadapt软件仅保留20bp的sgRNA部分

合并kmer

files=list.files(pattern=”tsv”)
kmers =list()
meta=read.table(“../../../library.csv”,sep=”,”,col.names=c(“sgRNA”,”seq”,”gene”))
for (i in 1:6) {kmers[[i]] = read.table(files[i],col.names=c(“seq”,”counts”)) }

counts = merge(meta,kmers[[1]],by=”seq”,all.x=T,sort=F)
colnames(counts)[length(colnames(counts))] = files[1]
for (i in 2:6) {counts = merge(counts,kmers[[i]],by=”seq”,all.x=T,sort=F);colnames(counts)[length(colnames(counts))] = files[i] }

write.table(counts[,-1],”counts.txt”,quote=F,row.names=F,sep=”\t”)
